# Access Control Map

## Free pages

These files are free and do not require a paid subscription:

```txt
model 1.html
model 1-ar.html
```

## Private pages

All other model/revision part files are private and protected by:

```txt
js/access-control.js
```

Protected files include:

```txt
model 2.html → model 10.html
model 2-ar.html → model 10-ar.html
model-10-ar.html
01_transition_elements.html
02_chemical_analysis.html
03_chemical_equilibrium.html
04_electrochemistry.html
05_organic_chemistry.html
06_essay_questions.html
07_Calculation problems.html
transition-elements-ar.html
chemical-analysis-ar.html
chemical-equilibrium-ar.html
electro-chemistry-ar.html
organic-chemistry-ar.html
essay-questions-ar.html
Calculation problems-ar.html
and Arabic duplicate/private part versions
```

## Rule

- Not logged in → redirected to login page.
- Logged in but free plan → redirected to pricing page.
- Monthly/VIP/Admin/Teacher → access allowed.

## Important

This is browser-level protection for the current static site. For production, use Supabase Row Level Security + signed storage URLs for true server-side protection.
